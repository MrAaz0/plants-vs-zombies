# Plants vs Zombies

This project is a recreation of the classic tower defense game, Plants vs Zombies. Players must strategically place plants to fend off waves of zombies, utilizing different plants and defensive strategies to survive as long as possible.

## Features
- Various types of plants, each with unique abilities
- Multiple levels and increasing difficulty
- Fun animations and graphics
- Aiming for a balance between strategy and entertainment

## Getting Started
To get started with the project, clone the repository and run the main application file. Make sure to follow the instructions in the upcoming sections to set up your development environment.
